# UML Process Extractor

A web application that transforms natural language process descriptions into visual UML/BPMN diagrams, entity highlighting, and resource flow analysis.

## Features

- **Natural Language Processing**: Extract entities (Agents, Tasks, Conditions) from text
- **BPMN Diagram Generation**: Automatic PlantUML diagram generation
- **Entity Highlighting**: Visual color-coded entity recognition
- **Resource Flow Analysis**: Heatmap and Sankey diagrams showing agent interactions
- **Test Examples**: Pre-loaded example texts for quick testing
